# Tinder for movies app

It works like tinder, swipe left or right.
It has no API, it just fetches some movies from [themoviedb](https://developers.themoviedb.org/3).

## How to run localy?

```
npm install
npm run start
```

## Technologies

Project created with:

- React
- TypeScript
- Redux
- Redux-thunk
- Axios
- React-router-dom
- React-tinder-card
- Sass
